# Security Policy

Please report vulnerabilities privately through GitHub private vulnerability reporting when enabled for this repository. If that is unavailable, contact the maintainers without publishing exploit details.

Do not include live Remnawave root tokens, RemnaGuard raw tokens, subscription URLs, or production request bodies in reports.

Supported compatibility target:

| RemnaGuard | Remnawave API |
| --- | --- |
| v1 baseline | 2.7.4 |

RemnaGuard denies unsupported and mismatched upstream versions by default.
